> 将危险表达改为可播表达，所有改写均经过实际直播验证或基于平台规则白名单
> 来源：逻辑对齐稿验证样本 + 腾讯广告审核规则 + 历史封禁改写经验

## 使用方式

当审核发现违规时，按**一级标签→二级标签**定位到对应条目，直接取"安全改写"使用。

## 六、改写原则速查

### 核心替换映射表

| ❌ 危险方向 | ✅ 安全方向 |
|------------|-----------|
| 赚钱/收入/变现 | 兴趣/爱好/提升/丰富 |
| 接单/录制/角色 | 了解/体验/尝试/方向 |
| 荷包/口袋/收获 | 魅力/能力/进步/成长 |
| 喜马/官方/平台 | 行业内/品牌/课程配套 |
| 抢/限/最后/仅 | 本期/当前/有限 |
| 包会/保证/一定 | 可以/有机会/看个人努力 |
| 安排/匹配/帮助找 | 介绍/了解/包含/有路径 |
| 送/免费/额外 | 包含/福利/其中/体验 |

### 改写三原则

1. **去结果化**：不说学了能得到什么结果，说学的过程和价值
2. **去具体化**：不说具体金额/时间/数量，说大致范围和可能性
3. **去交换化**：不说"做X就能得到Y"，说"X里面包含Y"
