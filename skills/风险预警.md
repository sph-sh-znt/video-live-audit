# 风险预警（三级预警系统）

> 功能定位：对接审核结果，按违规严重程度触发三级预警，统一处理规范
> 核心价值：让主播/中控/运营快速判断"这个违规有多严重，该怎么处理"

---

## 三级预警体系

### 🟢 一级预警（轻度风险）

| 属性 | 说明 |
|------|------|
| **触发条件** | 命中L-xxxx（轻度违规）规则 / 确定性表达占比10-20% / 单个低风险词 |
| **违规特征** | 非硬性违规，但可能影响用户体验或降低推荐量 |
| **AI行为** | 弹窗提醒 + 标记记录 |
| **主播/中控处理** | 当场知晓，可继续直播，事后记录优化 |
| **示例** | "总比刷手机强"（贬低对比）、封面文字略带诱导、画面轻微模糊 |

**处理SOP：**
1. AI弹窗/标注提示："⚠️ [一级提醒] 发现轻度风险表述，建议后续优化"
2. 中控在备忘录记录
3. 直播结束后统一修改话术
4. 无需中断直播

---

### 🟡 二级预警（中度风险）

| 属性 | 说明 |
|------|------|
| **触发条件** | 命中M-xxxx（中度违规）或B-04/B-06/B-070（中度业务风险） / 语义逻辑链部分命中 / 利益诱导动作 |
| **违规特征** | 易触发限流、警告，但不至于直接封禁 |
| **AI行为** | 弹窗提醒 + 语音预警（如有中控配合） |
| **主播/中控处理** | **立即停止当前话术/行为，3分钟内整改** |
| **示例** | "扣已加安排声音鉴定"（利益诱导）、低门槛人群圈定组合、封建迷信内容 |

**处理SOP：**
1. AI弹窗+语音提示："🟡 [二级警告] 检测到中度风险——[具体违规类型]，请立即调整"
2. 中控立即在评论区发提示信号（约定好的暗号）
3. 主播在3分钟内切换到备用安全话术
4. 中控记录：违规时间点、原话术、替换话术、恢复时间
5. 直播后纳入复盘

---

### 🔴 三级预警（重度风险/红线）

| 属性 | 说明 |
|------|------|
| **触发条件** | 命中S-xxxx/H-xxxx/B-01~B-03/B-05/B-05（重度业务风险） / 语义逻辑链完整命中 / 绝对化用语+收益表达组合 |
| **违规特征** | 平台红线，直接触发封禁 |
| **AI行为** | **强制弹窗拦截** + 连续警报 |
| **主播/中控处理** | **中控立刻叫停直播内容，整改完成方可继续** |
| **示例** | "想不想用声音创造价值的""学完去录主角""月入XXXX""喜马官方""招主播" |

**处理SOP：**
1. AI强制弹窗+连续警报："🔴 [三级拦截] 检测到严重违规——[具体违规类型]，立即停止！"
2. 中控**立即**打断主播（约定紧急打断方式）
3. 主播**立即停止当前话题**，切换到绝对安全的中立话题（如干货教学/互动问答）
4. 如已触发平台处罚，中控立即记录封禁信息
5.本场直播结束后必须进行复盘溯源

## 预警等级判定逻辑

```
输入审核命中的所有违规项
    ↓
取最高违规等级作为基准
    ↓
┌─ 存在S/H类违规 ──→ 🔴 三级（不可降级）
│
├─ 存在完整Chain-01~03命中 ──→ 🔴 三级
│
├─ 存在B类重度+数量≥2 ──→ 🔴 三级
│
├─ 存在M类违规 或 B类中度 ──→ 🟡 二级
│   特殊情况：单一M类且语境安全 → 可降至🟢一级
│
└─ 仅存在L类违规 ──→ 🟢 一级
```

## 实时预警接口设计

### 输入格式（实时流式）

```json
{
  "type": "live_stream_audit",
  "stream_id": "直播间ID",
  "timestamp": "ISO时间戳",
  "text_segment": "当前说话片段",
  "context": {
    "current_section": "当前环节",
    "previous_segments": ["前3个片段"]
  }
}
```

### 输出格式

```json
{
  "alert_level": "L1/L2/L3",
  "alert_type": "违规类型名称",
  "rule_id": "规则编号",
  "matched_text": "命中的原文",
  "suggestion": "合规替代建议",
  "action_required": "提醒记录/立即整改/强制拦截",
  "safe_alternative": "可用的安全替代话术"
}
```

## 安全话术库（应急使用）

当触发🟡二级或🔴三级预警时，主播可立即切换以下安全话题：

### 干货教学类（永远安全）
- "来，大家跟我一起做一个发声练习"
- "我们来讲一个气息控制的小技巧"
- "大家知道为什么有的人声音好听有的人不好听吗？其实关键在于..."
- "我给大家演示一下怎么读这段文字更有感情"

### 互动问答类（永远安全）
- "直播间有没有谁之前学过配音的？打个'学'字我看看"
- "大家觉得最难的是哪一部分？打在公屏上"
- "刚才进来的新朋友，你们最想了解什么？"

### 中立过渡类
- "我们稍作休息，接下来给大家讲一个新的知识点"
- "说到这里，我想起一个很有意思的事情..."

## 预警统计与报表

### 单场统计维度

| 维度 | 指标 |
|------|------|
| 总预警次数 | L1:__次 L2:__次 L3:__次 |
| 最高预警等级 | 🟢/🟡/🔴 |
| 平均响应时间 | 从预警发出到主播整改的秒数 |
| 触发TOP3违规类型 | 排名前三的违规类型 |
| 整改完成率 | 预警后成功整改的比例 |

### 周度/月度趋势

| 维度 | 分析目标 |
|------|---------|
| 违规类型分布变化 | 哪类违规在增加/减少 |
| 同一违规重复率 | 是否同一问题反复出现 |
| 预警准确率 | AI预警是否被人工确认为真实风险 |
| 封禁率 vs 预警覆盖率 | 封禁场次中有多少被AI提前预警到了 |
