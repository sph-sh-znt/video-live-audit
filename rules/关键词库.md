## 九、关键词检测策略说明

### 9.1 三级检测机制

```
Level 1: 精确匹配（字符串完全一致）
  ↓ 未命中
Level 2: 模糊匹配（包含/近义词/变体）
  ↓ 未命中
Level 3: 语义分析（整段话逻辑链判断）
```

### 9.2 上下文敏感词

以下词汇**单独出现不一定违规**，需要结合上下文判断：

| 词汇 | 安全场景 | 危险场景 |
|------|---------|---------|
| 免费 | "免费试听""免费公开课"（真实活动） | "免费接单""免费赚钱渠道" |
| 官方 | "官方课程资料"（自研课程） | "官方渠道""平台官方提供" |
| 1元/一米 | "1元体验课"（真实价格） | "投1元赚100元" |
| 加我/加上 | "有问题可以加我咨询" | "加上我带你赚钱" |
| 机会 | "学习是个好机会" | "这是个赚钱的好机会" |
| 改变 | "改变自己的发音习惯" | "改变你的未来生活" |
| 价值 | "这个知识点很有价值" | "用声音创造价值" |
| 回报 | "付出总有回报"（励志语境） | "投入1米获得回报"（商业语境） |
| 收获 | "收获知识收获成长" | "一定会收获颇丰"（收益语境） |
| 匹配 | "匹配适合自己的学习方法" | "根据你的声音给你匹配类型"（平台承诺） |
| 安排 | "安排一下学习计划" | "我来帮你安排声音鉴定"（利益交换） |
| 福利 | "今天的课程福利" | "报名就送声音鉴定福利"（利益诱导） |
