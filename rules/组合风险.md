# 组合风险场景

> 单个表达可能不违规，但多个表达组合在一起时风险急剧升高
> 来源：逻辑对齐稿 + 历史封禁数据分析 + 语义链路检测

---

## 核心原则

**"1+1 > 2"效应**：两个🟡中 risk的表达连在一起 = 🔴高 risk

平台审核不是逐句判罚的，而是**看整段话传递的最终含义**。组合风险就是捕捉这种"整体大于部分之和"的场景。

---

## 四大致命组合

### 组合1：低价体验课 + 副业收益暗示 + 私域承接

**风险等级：🔴 红线（任何账号状态）**

```
典型话术模式：
"1元学5天课程（低价体验课）
 → 学完可以接单/赚钱/获得收益（副业收益暗示）
 → 加上助教老师微信，老师带你（私域承接）"
```

**为什么致命：**
- 低价入口降低用户防备
- 收益暗示激发用户行动欲望
- 私域承接完成转化闭环
- **三者合在一起 = 平台判定为"以虚假承诺诱导交易"的完整证据链**

**真实案例触发点：**
- "1米学全程5天" + "用声音创造价值" + "加上助教回来扣已加"
- "今天给到大家的是福利名额" + "口袋塞得越来越满" + "长按识别添加助教"

**安全做法（三去一）：**
- ✅ 保留低价体验课（真实价格即可）
- ❌ 删除收益暗示 → 改为"提升自己/兴趣爱好"
- ⚠️ 弱化私域承接 → 改为"按提示操作添加助教"

### 组合2：刚禁播恢复账号 + 荷包/收入/接单表达

**风险等级：🔴 线线（正常账号可能是🟡中危）**

```
典型场景：
账号状态：近7天内解禁恢复 / 近30天内有封禁记录
话术中出现：
 - "荷包""收获""获得""口袋""小果果"
 - 或 "接单""录制""角色""机会"
 - 或 "收入""赚钱""变现""创收"
```

**为什么致命：**
- 平台对恢复期账号有**自动升级处罚**机制
- 同样的"荷包更丰满"，正常账号=弹窗提醒，恢复期账号=中断直播
- 恢复期账号的容错空间≈0

**处理规则：**
| 账号状态 | 遇到收益表达时的处理 |
|---------|-------------------|
| 🟢 正常 | 🟡高危，建议修改 |
| 🟡 观察期 | 🔴红线，必须删除 |
| 🟠 高危期 | 🔴红线，必须删除 |
| 🔴 恢复期 | 🔴红线，**整段重写** |

### 组合3：师资强背书 + 平台资源暗示 + 逼单

**风险等级：🔴 红线**

```
典型话术模式：
"张震老师/大咖老师/XX年经验（师资强背书）
 → 官方资源/平台提供/我们帮你匹配（资源暗示）
 → 名额有限/仅限XX名/马上结束（逼单）"
```

**为什么致命：**
- 强背书建立信任 → 用户相信你说的是真的
- 资源暗示让用户觉得"有门路"
- 逼单推动用户立即行动
- **三者结合 = "权威机构帮你走后门+限时优惠" = 虚假宣传+诱导交易双重违规**

**真实案例触发点：**
- "配音界的大咖老师...代表作大家都看过吧...不用我多说了" + "我们帮你挑选" + "仅限20个名额"
- "三位大咖老师的课...反复学反复练" + "老师会给你说的" + "报完咱就准备结束"

**安全做法：**
- 师资介绍✅保留（头衔真实可验证即可）
- 资源暗示❌改为"课程里会介绍各方向"
- 逼单⚠️ 弱化或删除紧迫感表述

### 组合4：静态画面 + 大面积营销文字 + 低互动

**风险等级：🟡高危~🔴红线（触发低质量内容判定）**

```
典型场景：
- 主播不出镜/长时间展示PPT或海报
- 画面上堆砌大量营销文字（价格/福利/二维码/引导语）
- 主播不互动或低互动（只念稿/录播音频）
- 用户进来看到的就是一张"广告图"
```

**为什么危险：**
- 触发《低质量内容认定细则》L-012~L-022多条
- 平台判定为"非实质内容直播/博眼球/录播替代"
- 可能导致减少推荐→禁播→封禁的递进处罚

**安全做法：**
- 主播必须出镜且保持互动
- 营销文字面积<20%画面
- 每2分钟必须有实质性内容输出（干货/互动/演示）
- 封面和贴片不要同时堆砌营销信息

## 其他高频组合（持续补充）

### 组合5：宝妈人群圈定 + 低门槛 + 结果暗示

**风险等级：🟡高危（投放）/ 🟡中危（自然流）**

```
"在家带娃的宝妈（圈定人群）
 → 不用露脸/在家就做得到/时间灵活（低门槛）
 → 学了就能XX/给自己一个机会（结果暗示）"
```

### 组合6：焦虑制造 + 紧迫感 + 唯一解决方案

**风险等级：🟡高危**

```
"很多人一直在想但没迈出去（焦虑）
 → 就这一步拉开了差距（加剧焦虑）
 → 名额有限/马上结束/错过没了（紧迫感）
 → 我们的课程就是答案（唯一解）"
```

### 组合7：演绎剧本 + 虚假人设 + 利益引导

**风险等级：🔴红线**

```
编造/夸大学员经历（虚假人设）
 → "我以前也是...直到学了..."（剧本演绎）
 → "所以你们也应该...报名..."（利益引导）
```

## 组合检测算法

在审核流程中增加STEP：组合风险扫描

```
对已标记的所有风险项：
1. 计算两两之间的距离（相邻句内/同段落内/同一环节内）
2. 检查是否命中上述4大致命组合模式
3. 如命中 → 在原有风险等级基础上再升一级
4. 输出组合风险警告："检测到[组合X]模式，多个中风险表达聚集"

示例检测结果：
┌─────────────────────────────────────────────┐
│ ⚠️ 组合风险警告                              │
│ 检测到 [组合1: 低价+收益+私承] 模式          │
│                                             │
│ 参与表达：                                   │
│   [第3句] "1米学全程5天" (低价体验课)         │
│   [第8句] "用声音创造价值" (收益暗示)         │
│   [第12句] "加上助教回来扣已加" (私域承接)    │
│                                             │
│ 三段话分布在逼单环节(相距4句内)，构成完整诱导链 │
│ 建议：至少删除其中1-2个风险表达               │
└─────────────────────────────────────────────┘
```
