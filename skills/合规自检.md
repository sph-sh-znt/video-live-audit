# 合规自检（全量扫描模块）

> 功能定位：对直播全要素（脚本+封面+贴片+布景图+标题）进行综合合规检查
> 与"话术预审"的区别：话术预审专注文本，合规自检覆盖直播全部可见元素

---

## 使用方式

```
/视频号审核:合规自检 <检查内容> [检查模式：快速/完整]
```

**检查内容支持：**
- 纯文本（话术/文案）
- 文件路径
- 结构化描述（封面描述+贴片描述+场景描述+话术文本）

不指定模式时默认"快速模式"。

---

## 检查清单（视频号直播全要素）

### A类：文本内容检查（继承话术预审能力）

| 编号 | 检查项 | 检查方法 | 对应规则 |
|------|--------|---------|---------|

| A-02 | 收益/赚钱表达扫描 | 含隐喻暗语检测 | 关键词库·二 |
| A-03 | 虚假营销表达扫描 | 紧迫感/低门槛/利益诱导 | 关键词库·三 |
| A-04 | 品牌/平台违禁词扫描 | 喜马化检测 | 关键词库·四 |
| A-05 | 招聘/引流词扫描 | 站外引流检测 | 关键词库·五 |
| A-06 | 效果承诺扫描 | 承诺类表达 | 关键词库·六 |
| A-07 | 语义逻辑链分析 | 5大高风险链路 | 违规规则库·五 |
| A-08 | 上下文敏感词判断 | 安全/危险语境区分 | 关键词库·九 |

### B类：视觉元素检查

| 编号 | 检查项 | 检查标准 | 对应规则 |
|------|--------|---------|---------|
| B-01 | 封面文字 | 无诱导性/噱头性文字；无违规关键词 | L-015, H-038 |
| B-02 | 封面图像 | 无色情/暴力/低俗/恐怖画面 | S-010~S-017, H-010~H-026 |
| B-03 | 贴片/字幕 | 品牌名位置合规（左上方≥25px）；无大面积堆砌文字 | L-021, 广告规范 |
| B-04 | 直播背景 | 无不当场所（酒吧/网吧/卧室暧昧等）；无违规物品 | L-032, H-019 |
| B-05 | 主播服装 | 不触犯H-050~H-055任何一条服装违规 | H-050~H-055 |
| B-06 | 画面质量 | 无黑屏/抖动/卡顿/模糊低清/颠倒旋转 | L-017~L-020 |
| B-07 | 特殊标识 | 无国旗国徽军警制服等不当使用 | S-009, H-050 |
| B-08 | 二维码/联系方式 | 非必要不展示；展示方式需合规 | L-030 |

### C类：行为规范检查

| 编号 | 检查项 | 检查标准 | 对应规则 |
|------|--------|---------|---------|
| C-01 | 是否真人直播 | 非录播/非无人直播/非虚拟人代替 | L-012~L-014 |
| C-02 | 互动频率 | 语音直播需有实时互动，不能全程录制音频 | L-014 |
| C-03 | 抽烟/饮酒 | 直播间无抽烟无饮酒酗酒内容 | L-033~L-034 |
| C-04 | 驾驶安全 | 无边驾驶边直播 | M-023 |
| C-05 | 未成年人 | 无未成年人出镜/连麦/当模特 | M-010~M-013 |

### D类：业务线专项检查

| 编号 | 检查项 | 检查标准 |

## 执行流程

```
输入检查内容
    ↓
判断输入类型（纯文本/文件/结构化描述）
    ↓
┌─ 快速模式 ─────────────────────────────┐
│  仅执行A类（文本）+ B类关键项（B-01/B-05）│
│  耗时：约30秒                            │
└─────────────────────────────────────────↓
┌─ 完整模式 ─────────────────────────────┐
│  执行A+B+C+D全部检查项                   │
│  耗时：约1-2分钟                         │
└─────────────────────────────────────────↓
汇总所有检查结果
    ↓
交叉验证（去除重复告警）
    ↓
按预警等级排序输出报告
```

## 快速模式 vs 完整模式对比

| 维度 | 快速模式 | 完整模式 |
|------|---------|---------|
| 适用场景 | 上线前快速自查 | 正式提审/月度全量检查 |
| 检查范围 | 文本+封面+服装 | 全要素 |
| 耗时 | ~30秒 | ~1-2分钟 |
| 输出详度 | 仅列出问题项 | 问题项+修改建议+修改后全文 |
