# 给王佳、吴桐、淑珍的AI审核体使用教学话术

> 使用场景：于彤/晓静在群里或会上口头讲解时用
> 也可以直接发文字版给她们自学

---

## 开场（3人一起说）

各位，咱们视频号直播AI审核体已经搭好了，今天教大家怎么用。

这个系统干的事情很简单：**你把话术发给它，它告诉你哪里会违规、怎么改。**

以前咱们人工自查一场直播要60分钟，现在用这个AI，**15分钟搞定，而且比人查得更全。**

---

## 第一步：怎么用（所有人必学）

### 最简单的用法，就三步：

**第一步：找到你的AI工具**
- 你们用哪个AI都行——Claude、ChatGPT、通义、豆包都可以
- 打开之后，先把下面这段话发过去（只发一次，以后就不用发了）：

```
请你扮演一个【视频号直播合规审核助手】。
请先读取以下规则文件的内容并记住：
1. 违规规则库（包含S/H/M/L/B五级162条规则）
2. 关键词库（包含约270个违禁词/敏感词）
3. 业务线差异化规则（有声投放/IP自然流/IP投流）

规则读完后，帮我审核直播话术。
```

**第二步：粘贴你要审的话术**
- 把你的直播脚本、逐字稿、或者某一段话直接粘贴进去
- **记得告诉它是哪条业务线的！** 这一点很重要，因为三条线标准不一样

**第三步：看结果，照着改**
- AI会告诉你哪些地方🔴必须改、哪些🟡建议改
- 每条都会给你"改成这样说"
- 你照着改就行

**你的工作流程变化：**

### 🎯 杨淑珍（项目总统筹）

淑珍姐，您主要是**统筹和最终把关**，不需要每句话都自己去跑AI。

**您的使用场景有三个：**

**场景1：周会/阶段验收会时**
让大家把当周的审核报告发出来，您看总判定这一行就够了：
- 🟢 = 通过，没问题
- 🟡 = 有风险但可控，让运营说明怎么处理的
- 🔴 = 有问题，让运营重改后再提交

**场景2：两条线规则有分歧时**
如果佳姐和吴桐对同一条话术的判定不一致，把那段话发给我，我帮您出一份**仲裁意见**，标注哪条线为什么判得不一样。

**场景3：新规则/新封禁案例出现时**
运营发现新的违规类型 → 反馈给技术侧 → 我们24小时内更新规则库 → 您确认后生效

**您只需要记住一个原则：**
> **🔴红线三条线通用（喜马/招聘/极限词），🟡高危各线不同，🟢低危看账号状态。**

## 第三步：遇到问题怎么办

| 问题 | 怎么办 |
|------|--------|
| AI判错了（明明没问题说有问题） | 截图发给于彤/晓静，标注"误判"，我们1个工作日内核实 |
| AI漏判了（播了之后才发现有问题） | 把原话和处罚通知发给我们，我们补进规则库 |
| 不确定某个表达能不能说 | 发给AI问："[这句话]在[某业务线]下能不能用？" |
| 想要完整修改版 | 把原文发给我，我输出逐句修改后的版本 |
| 新增了封禁案例 | 按"复盘溯源"格式提交，我们自动更新规则 |
