# 三业务线差异化审核规则

> 最后更新：2026-06-22
> 核心原则：**投放直播过双审（内容+广告），自然流只过内容审**

---

## 一、三条业务线对比总览

| 维度 | 有声线·投放直播 | IP线·自然流 | IP线·投流直播 |
|------|---------------|------------|-------------|
| **审核通道** | 平台内容审 + **腾讯广告审** | 仅平台内容审 | 平台内容审 + **腾讯广告审** |
| **审核严格度** | ★★★★★ 最严 | ★★★☆☆ 相对宽松 | ★★★★☆ 严格 |
| **品牌露出** | ❌ 禁止"喜马"任何形式 | ✅ 可用"张震工作室" | ✅ 可用"张震老师" |
| **收益表达** | ❌ 零容忍 | ⚠️ 兴趣导向为主，少量可接受 | 🟡 广告审核有明确白名单 |
| **核心风险** | 品牌违规+利益诱导+招聘嫌疑 | 引流方式+连麦合规 | 虚假紧迫感+广告法违规 |
| **话术风格** | 强转化、逼单密集 | 弱转化、重连麦互动 | 中强转化、有框架结构 |
| **典型场景** | 卖1元课→加微→声音鉴定 | 连麦鉴定→点关注→加助教 | 卖1元课→卡片→加助教 |

## 二、各业务线专属规则

### 2.1 有声线 · 投放直播（最严格）

**身份特征：**
- 主播代表：喜播教育/喜马拉雅生态
- 课程产品：5天22节声音美化课（1米）
- 目标人群：宝妈、上班族、银发
- 转化路径：直播间报名 → 加助教 → 微信群 → 正价课

| AY-02 | **声音鉴定利益交换** | "报名就送声音鉴定""扣已加安排鉴定""打已加领福利" | 🔴三级 | 鉴定作为报名交换条件=利益诱导 |

### 2.2 IP线（张震）· 自然流直播

**身份特征：**
- 主播代表：张震工作室 / 张震老师个人IP
- 课程产品：张震亲授有声课程（5天+声音鉴定）
- 目标人群：宝妈/上班族/声音爱好者/想提升表达力的人
- 转化路径：连麦鉴定 → 点关注主页 → 扫码加助教 → 报名

#### 专属规则

| 编号 | 规则项 | 说明 | 风险等级 |
|------|--------|------|---------|
| IPN-01 | **连麦内容合��** | 连麦时不能做销售转化，只能做干货指导+轻度引导 | 🟡二级 |
| IPN-02 | **引流路径合规** | "点左上角头像→主页第一条→截图扫码"此路径本身合规 | 🟢一级 |
| IPN-03 | **二维码/微信号展示** | 自然流不禁止展示联系方式，但不能用"加微/私聊我"等诱导性用语 | 🟡二级 |
| IPN-04 | **张震IP使用** | "张震老师""张震工作室"可用；头衔需真实（如"配过《疯狂动物城》狐尼克"） | 🟢一级 |
| IPN-05 | **声音鉴定定位** | 自然流中"声音鉴定"作为**连麦服务内容**而非**报名交换条件**时合规 | 🟢一级 |
| IPN-06 | **收益表达边界** | 自然流可适度提及"丰富业余生活""多一个兴趣方向"；不能提金额/赚钱/接单 | 🟡二级 |
| IPN-07 | **连麦时长控制** | 单人连麦教学不超过3分钟，避免长时间无互动导致低质量判定 | 🟡二级 |
| IPN-08 | **低质内容风险** | 无人连麦时长时间等待/重复相同内容 → 可能触发L-017黑屏挂机/ L-016猎奇无实质内容 | 🟡二级 |

#### IP自然流 vs 有声投放的关键差异

| 表达 | 有声投放 | IP自然流 | 差异原因 |
|------|---------|---------|---------|
| "声音鉴定" | ❌ 作为报名福利 | ✅ 作为连麦服务 | 定位不同 |
| "加助教领鉴定" | 🔴 利益诱导 | ⚠️ 需弱化连接 | 语境不同 |
| "点头像→主页→扫码" | 🟡 需注意措辞 | ✅ 自然流标准引流路径 | 审核通道不同 |
| "张震老师" | ✅ 可用 | ✅ 核心 | 都可用 |
| "丰富业余生活" | ✅ 可用 | ✅ 核心话术 | 自然流主推方向 |
| "提升自己/兴趣爱好" | ✅ 可用 | ✅ 核心话术 | 共用安全表达 |
| "不用出门/在家就能学" | 🟡 低门槛组合风险 | ✅ 自然流可适度使用 | 投放审核更严 |

#### IP自然流安全话术模板（可直接用）

**欢迎留人（安全版）：**
> "欢迎新进直播间的同学，咱们是张震工作室帮你做声音鉴定的直播间。不知道自己声音怎样的、适合做什么的？可以上来连麦，我听听你的声音，给你一些方向建议。"

**痛点种草（安全版）：**
> "宝妈们，你想过吗——等娃下午睡熟了，你拿出手机，戴上耳机，录一段温暖的儿童故事。花一份时间，既陪了孩子，又提升了自己。"

**连麦引导（安全版）：**
> "大家都听到我的声音了，也让我听听大家的声音。觉得自己声音不自信的，上麦连线和老师交流一下，我指出你的问题所在。"

**轻量引导（安全版）：**
> "感兴趣想继续深入学习的同学，可以点左上角头像看主页第一条截图，就能跟着张震老师学习有声课程。"

### 2.3 IP线（张震）· 投流直播

**身份特征：**
- 主播代表：张震老师 / 冬冬师姐（助教）
- 课程产品：张震亲授5天课 + 声音鉴定(价值199) + 资料包(99)
- 目标人群：宝妈/上班族/想额外收获的人群
- 转化路径：点击白色卡片 → 加助教 → 微信 → 正价课

#### 专属红线

| 编号 | 违规类型 | 高危表达示例 | 风险等级 | 说明 |
|------|---------|-------------|---------|------|
| IPI-01 | **口袋/获得感过度** | "口袋塞得越来越满""获得一些额外收获""取得对应收获" | 🔴三级 | 收益隐喻，投流广告审核重点打击 |
| IPI-02 | **价值锚定虚高** | "价值199的声音鉴定""价值99的资料包" | 🟡二级 | 需真实价值佐证 |
| IPI-03 | **紧迫感虚假** | "仅限20个名额""张震老师中午就直播了你就没法听" | 🟡二级 | 腾讯广告禁止虚假限时限量 |
| IPI-04 | **信息差叙事** | "你和别人的差距就是一个信息差" | 🟡二级 | 焦虑制造+暗示他人已获利 |
| IPI-05 | **需求夸大** | "把你们阅历优势放大以实现获得感" | 🟡二级 | "获得感"=收益暗语 |
| IPI-06 | **接书路径暗示** | "告诉你接书的路径在哪里" | 🔴三级 | 投流中直接提"接书"=兼职暗示 |
| IPI-07 | **播放量=收益关联** | "你的播放量越高，收获也就越高" | 🔴三级 | 直接建立投入产出关系 |
| IPI-08 | **女性优势变现暗示** | "女声的需求很多！这个行业就是普通人入局的好机会" | 🟡二级 | 暗示入局可获得收益 |
| IPI-09 | **卡片引导频率过高** | 每段话术后都"(点击卡片、手机展示、添加助教)" | 🟡二级 | 过度营销感，影响用户体验 |
| IPI-10 | **见面礼/福利诱导** | "先送同学们一个见面礼""只要今天...就可以领取" | 🟡二级 | 利益诱导框架 |

#### IP投流 vs 有声投放的差异

| 维度 | 有声投放 | IP投流 |
|------|---------|--------|
| 品牌 | 不能提喜马 | 可以提张震 |
| 师资背书 | 艾宝良/武向彤等外部大咖 | 张震本人核心+外部辅助 |
| 产品包装 | "声音美化课程" | "讲故事讲小说训练" |
| 福利设计 | 声音鉴定 | 声音鉴定+资料包（双福利=更高诱导风险） |
| 话术密度 | 中等 | **很高**（每段都插卡片引导） |
| 最大风险 | 哜牌违规+利益诱导 | **收益隐喻+虚假紧迫感** |
| 广告审核重点 | 喜马化残留+兼职暗示 | 获得感/收获/口袋/路径 等暗语 |

#### IP投流白名单（腾讯广告审核明确允许）

| 表达 | 条件 |
|------|------|
| "0基础学配音/讲小说" | 职业培训品类允许 |
| "不限年龄/学历/不用真人出镜" | 单独使用安全 |
| "方言讲故事/讲小说方向" | 真实存在的学习方向 |
| "5天张震亲授直播课+录播课" | 真实课程安排 |
| "科学发声/让声音变好听" | 教学内容描述 |
| "虚拟受益人剧情演绎" | 不冒充真实学员 |

## 三、跨线共用规则

以下规则**三条线都必须遵守**：

### 3.1 绝对三不管（谁说都封）

| 类别 | 词汇/表达 |
|------|----------|
| 极限词 | 最好/第一/唯一/顶级/绝对/百分百/无敌/完美 |
| 喜马系 | 喜马拉雅/喜马/喜马官方/喜马logo/喜马生态 |
| 招聘系 | 招主播/招募/岗前培训/入职/兼职(投放)/副业(投放) |
| 具体收益 | 月入XX/一小时赚XX/日薪/时薪/创收(视频号) |
| 色情暴力 | （通用，所有平台） |

### 3.2 分线放宽规则

| 规则 | 有声投放 | IP自然流 | IP投流 |
|------|---------|---------|--------|
| 提及"机会" | 🟡 需看语境 | ✅ 较宽容 | 🟡 需看语境 |
| 提及"改变/提升" | 🟡 不能接"未来/生活" | ✅ "改变自己"OK | 🟡 不能接"收入/生活" |
| 提及"鉴定/测评" | ❌ 不能当交换条件 | ✅ 连麦服务可用 | ❌ 不能当报名福利 |
| 展示联系方式 | ❌ 禁止"加微/私聊" | ✅ 可展示二维码 | 🟡 需间接引导 |
| "官方"一词 | ❌ 任何形式禁用 | ⚠️ "张震工作室"可 | ❌ 禁用 |
| 低价/免费策略 | 🟡 "1米"需说明活动详情 | ✅ 相对宽松 | 🟡 需符合广告规范 |

## 四、审核时的业务线选择

### 使用方式

```
/视频号审核:话术预审 <话术文本> [业务线：有声投放/IP自然流/IP投流]
```

### 选择不同业务线后的差异行为

| 处理步骤 | 有声投放模式 | IP自然流模式 | IP投流模式 |
|---------|------------|------------|-----------|
| 加载规则集 | 全量S/H/M/L/B + AY全部 | 全量 + IPN全部 | 全量 + IPI全部 |
| 关键词权重 | 喜马词×5 / 收益词×5 / 鉴定词×3 | 喜马词×3 / 收益词×2 / 鉴定词×1 | 喜马词×3 / 收益词×5 / 暗语词×5 |
| 语义链灵敏度 | Chain-01~05 全部最高灵敏 | Chain-01~03 中等 / Chain-04~05 低 | Chain-01/03/05 最高 / Chain-02/04 中等 |
| 白名单适用 | 腾讯广告白名单 | 自然流宽松白名单 | 腾讯广告白名单+IP特有 |
| 输出报告标注 | 标注"[有声投放]专项" | 标注"[IP自然流]专项" | 标注"[IP投流]专项" |
