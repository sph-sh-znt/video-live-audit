## 项目背景

目前有声线、IP线视频号投放直播频繁触发平台违规封禁，各视频号直播运营对平台规则认知不一，人工事前自查漏检率高，缺乏标准化合规管控体系。

本智能体对标视频号官方直播规范，覆盖**事前脚本预审 → 事中风险预警 → 事后复盘溯源**全流程，目标：
- 短期：直播封禁率下降80%，人工自查耗时减少70%
- 中期：AI识别准确率≥95%
- 长期：形成"AI审核+人工复核+规则迭代"标准化体系

## 快速使用

### 第一次使用（3步上手）

```bash
# 1. 话术预审
/视频号审核:话术预审 "粘贴你的直播话术"

# 2. 查看报告中的🔴硬性违规，按建议修改

# 3. 修改后重新提交，确认🟢即可开播
```

### 日常使用流程

```
开播前：话术预审 → 修改 → 复查确认 → 开播
  ↓
直播中：AI实时监控 → 触发预警 → 中控配合 → 主播切换安全话题
  ↓
直播后：整理预警记录 → 常规复盘 / 封禁深度复盘 → 规则迭代
```
